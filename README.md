# COMP9311 SQL Project

This project contains SQL and PL/pgSQL solutions for database query tasks based on the MyMyUNSW database schema.

## Technologies
- PostgreSQL
- SQL
- PL/pgSQL

## Features
- Created SQL views for course, subject, staff, room, and enrolment queries
- Used joins, aggregation, filtering, grouping, and subqueries
- Implemented queries to retrieve academic staff, course information, pass rates, and student performance
- Practised relational database design and query optimisation concepts
